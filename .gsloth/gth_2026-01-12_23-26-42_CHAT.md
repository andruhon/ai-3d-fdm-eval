Based on the web search results, I can help you modify the OpenSCAD command for an isometric view from the bottom.

In OpenSCAD, the camera parameter format is:
```
--camera=translateX,translateY,translateZ,rotX,rotY,rotZ,distance
```

Or for the eye/center format:
```
--camera=eyeX,eyeY,eyeZ,centerX,centerY,centerZ
```

The current command `--camera=0,0,-100,0,0,0` positions the camera at (0,0,-100) looking at the center (0,0,0), giving a straight bottom-up view.

For an **isometric view from below**, you need to set appropriate rotation angles. According to [en.wikibooks.org](https://en.wikibooks.org/wiki/OpenSCAD_User_Manual/Transformations), OpenSCAD uses the right-hand rule for rotations:
- **rotX**: rotation about the X axis
- **rotY**: rotation about the Y axis  
- **rotZ**: rotation about the Z axis

For a standard isometric view from the **top**, you'd typically use rotations like `55,0,45` (or `35,0,45` for a more common engineering isometric).

For an isometric view from the **bottom** at 45 degrees, you need to flip the X rotation to look upward:

```bash
openscad -o output-bottom.png --render --backend=Manifold --camera=0,0,0,235,0,45 --autocenter --viewall rivet.scad
```

Or alternatively:

```bash
openscad -o output-bottom.png --render --backend=Manifold --camera=0,0,0,-55,0,45 --autocenter --viewall rivet.scad
```

**Explanation:**
- `0,0,0` — translation (centered, let `--autocenter` handle positioning)
- `-55` or `235` for rotX — tilts the view upward from below (opposite of the ~55° used for top isometric)
- `0` for rotY — no Y rotation
- `45` for rotZ — standard isometric rotation around Z axis

⚠️ **Note:** The exact angles depend on what "isometric at 45 degrees" means for your use case:
- If you want a true isometric projection (equal foreshortening on all axes), use approximately `54.736°` (or `arctan(√2)`) for the X tilt
- If you want a simpler 45° view angle, use `45` for rotX

For a simple 45° bottom view:
```bash
openscad -o output-bottom.png --render --backend=Manifold --camera=0,0,0,-45,0,45 --autocenter --viewall rivet.scad
```
