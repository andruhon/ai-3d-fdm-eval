# Development guidelines

This project uses Bun with TypeScript 5. We are building 3D modelling evaluation kit.

Entry point is in `index.ts`.

## OpenRouter TypeScript SDK

Read-only copy of used version of OpenRouter TypeScript SDK is available `readonly/typescript-sdk`.

### Documentation

Basic documentation form OpenRouter SDK is available in `scratchpad/openrouter-docs`.

More documentation is available in `readonly/typescript-sdk/docs`

## Requesting new tools and dependencies

Use `request_tool` to request new tool, use `request_dependency` to request new NPM (Bun) dependency.

Summery for the request will be presented. Wait for user's response after requesting tool or dependency.
